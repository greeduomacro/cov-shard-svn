    Installing
      1. Drag and drop the entire Blue Magic folder into your customs folder.

        If you need to install Magical Enchantments (Orb SVN already has it installed)
          1. Open "Blue Magic\Read Me\Scripts Edited" and open "AOS contains SA edits.copy".
          2. Merge the #region Enchantment/#endregion code into your own "Scripts\Misc\AOS.cs".
          3. Open "Blue Magic\Read Me\Scripts Support\MagicalEnchantments BM.rar" and place in your customs folder.

        If you need a copy of Mobile Abilities (Orb SVN already has it installed)
          1. Open "Blue Magic\Read Me\Scripts Support\Mobile Abilities BM.rar" and place in your customs folder.
          ~Note: This copy does not have any mobiles, it is just the core files.

        If you wish to install the Status System (not exactly using it anymore)
          1. Open "Read Me\Status Edits old.txt" and try to merge that into your core files.
            Good luck. This system presents a single file to edit multiple interactions within the base files many other custems interact with. Due to this, and the large number of changes RunUO does to these files it is almost impossible to provide a real guide.

      4. Compile and load world. In game use [Add BlueBen and place him where ever you wish.
      5. Add spawners for the monsters where ever you wish.
      6. Enjoy.
