    The BM command (stands for blue mage) can be used in a verity of ways.

    [BM
    Opens a menu displaying spells known. the blue dot can be clicked on to cast the spell and the question mark can be clicked on for a brief explanation of it.

    [BM [spellname], example [bm flamethrower
    Can be used to cast a spell though the command skipping the menu entirely. This is exceptionally useful as it allows you to set up key bindings (macros) for spells using the macro type Say.
    [BM Study
    Gives you a target in which you can click on a corpse to "study" it. Corpses of monsters that use Blue Magic can be studied to learn their spell. There is a chance you'll fail though.
    [BM Get
    Seer+ command, allows you to open a spell's known menu based on another player. Used to know if they know a spell or not (or even cast it your self). Currently there is no set command.

    [BM LearnAll
    Seer+ command, teaches you all learnable spells. GMs will have to learn spells like everyone else, this forces them to learn about the system, hopefully they'll do so increase someone pages them about something related to Blue Magic (such as where is my serpent mark?)

    [BM GiveItems
    Seer+, gives you a box containing various Blue Mage related items, meant to increase familiarity of obtainable equipment with your staff.

    [BM Cave
    Seer+ command, Teleports you to the cave dungeon added by the BME system in Ilshenar.

    [BM Log
    Admin+ command, opens a menu displaying a list of all accounts with characters that are Blue Mages and also lists their spells known. The list is updated after each world save.

    [BM Help
    Provides an in game explanation of the command arguments as a reminder.
